# FAQ Chatbot using NLP

## Project Overview

This project is a Frequently Asked Questions (FAQ) Chatbot built using Natural Language Processing (NLP). The chatbot answers user questions by finding the most similar question in the FAQ dataset using TF-IDF Vectorization and Cosine Similarity.

## Features

- Text Preprocessing
- TF-IDF Vectorization
- Cosine Similarity
- FAQ Question Matching
- Instant Response Generation
- Streamlit Web Application

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit

## Machine Learning / NLP Techniques

- TF-IDF Vectorization
- Cosine Similarity
